## Style Guidelines:

- Primary background: Cream (#FBF8F3) for a soft, luxurious feel.
- Primary text: Charcoal (#2C2C2C) for high readability and contrast.
- Accent color: Gold (#D4AF37) for primary buttons and highlights to signify luxury.
- Headline font: 'Georgia' (serif) for headings, providing an elegant and classic touch. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) for body text, offering a clean and modern reading experience. Note: currently only Google Fonts are supported.
- Responsive grid system for seamless adaptation across devices.
- Subtle fade-in animations on content load to enhance user experience.
